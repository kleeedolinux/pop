# Public Library API Examples

## Status of the sketches

These are architecture-level call-site contracts, not implemented APIs. Unless
a block says otherwise, it uses accepted Pop syntax: typed functions, records,
closures, `Result`, exhaustive `match`, prefix `try`, and lexical `defer`.
Postfix `?` remains optional-only. `await` and suspension-capable cleanup remain
unresolved language proposals.

## Installing official extensions

Official extensions are explicit Package dependencies, not standard imports:

```toml
[dependencies]
PopData = "0.1.0"
PopCli = "0.1.0"
```

```luau
using Pop.Data
using Pop.Cli
```

`Pop.Ai`, `Pop.Rpc`, `Pop.Syntax`, and `Pop.Lsp` use the same normal Package
flow. The local bootstrap manifests/builds exist now; registry installation via
`pop add` remains planned package-manager work.

## Text, bytes, sequences, and formats

Initial integer math is direct and allocation-free:

```luau
local lower = Math.min(first, second)
local upper = Math.max(first, second)
local magnitude = Math.abs(delta)
local divisor = Math.gcd(width, height)
local multiple = Math.lcm(width, height)
local direction = Math.sign(delta)
```

Sequence aggregates stay direct and do not materialize their inputs:

```luau
local hasInvalid = Sequence.any(values, function(value: Int): Boolean
    return value < 0
end)
local allReady = Sequence.all(jobs, isReady)
local total = Sequence.count(values)
local first = Sequence.firstOr(values, 0)
local selected = Sequence.countWhere(values, function(value: Int): Boolean
    return value > 10
end)
local page = Sequence.take(Sequence.drop(values, offset), pageSize)
local prefix = Sequence.takeWhile(values, function(value: Int): Boolean
    return value >= 0
end)
local combined = Sequence.concat(prefix, fallbackValues)
local totalCost = Sequence.sum(costs)
local batchSize = Sequence.product(dimensions)
local least = Sequence.minOr(values, 0)
local greatest = Sequence.maxOr(values, 0)
```

`any` and `all` stop as soon as their result is known. `count` consumes the
source and uses checked `Int` arithmetic. `firstOr` makes its empty fallback
explicit; `countWhere` checks its matching count. These terminal operations
allocate no storage of their own.

`sum` and `product` preserve checked left-to-right integer arithmetic. `minOr`
and `maxOr` use their fallback only for an empty source; no sentinel or dynamic
numeric protocol is involved.

`take`, `drop`, `takeWhile`, and `concat` are lazy. Each direct adapter
allocates one small state object but never materializes the input; nonpositive
counts have deterministic zero semantics. Pipelines can pass the returned
`Iterator<T>` directly into another Sequence operation.

Common text operations stay shallow and views avoid copies:

```luau
local clean: String = Text.trim(Text.normalize(input, Unicode.Form.Nfc))
local words: Iterator<Text.View> = Text.words(clean)
local names: List<String> = Sequence.collect(Sequence.map(words,
    function(word: Text.View): String
        return Text.toString(word)
    end))

local output: Result<Bytes, Bytes.Error> =
    Bytes.base64Decode(encoded, { maxBytes = 1_048_576 })
```

Typed JSON defaults to direct decoding rather than an intermediate tree:

```luau
match Json.decode(source, UserSchema, { maxDepth = 32 })
when Result.Ok(user) then
    print(user.name)
when Result.Error(error) then
    report(error)
end
```

Streaming and caller-owned buffers expose advanced control:

```luau
public function copyUsers(reader: Io.Reader, writer: Io.Writer): Result<(), Json.Error>
    local buffer = Bytes.buffer(64 * 1024)
    return Json.transform(reader, writer, buffer, function(user: User): User
        return redact(user)
    end)
end
```

Regex and glob are distinct, direct domains:

```luau
match Regex.compile("^[^@]+@[^@]+$")
when Result.Ok(email) then
    local valid: Boolean = Regex.isMatch(email, address)
when Result.Error(error) then
    report(error)
end
local sourceFile: Boolean = Glob.matches("src/**/*.pop", path)
```

## Locale and typed resources

YAML authoring input:

```yaml
# resources/en-US/app.yaml
welcome:
  params: { name: String }
  text: "Welcome, {name}"
files:
  params: { count: Int }
  plural:
    one: "{count} file"
    other: "{count} files"
```

Generated keys and parameter records keep runtime access typed:

```luau
match Resource.load(AppMessages, locale, resources)
when Result.Ok(catalog) then
    local text = Resource.text(catalog, AppMessages.welcome, { name = user.name })
    Terminal.writeLine(terminal, text)
when Result.Error(error) then
    report(error)
end
```

## Files and resource safety

The closure owns the file lifetime. `File.use` is a library function, not new
syntax, and closes on every return/panic path:

```luau
public function save(path: Path, value: AppSettings): Result<(), File.UseError<Json.Error>>
    return File.use(path, File.Mode.Write, files,
        function(file: File.Handle): Result<(), SaveError>
            return Json.write(file, value)
        end)
end
```

Large files use explicit streams and reusable storage:

```luau
local buffer = Bytes.buffer(128 * 1024)
local result = Io.copy(input, output, buffer, {
    maxBytes = 8 * 1024 * 1024 * 1024,
})
```

## Time, tasks, and cancellation

Accepted APIs pass cancellation explicitly:

```luau
local deadline = Time.after(clock, Time.seconds(5))
local options = { cancel = cancel, deadline = deadline }
local task: Task<Response> = Http.send(request, options)
```

Async work uses the accepted structured-ownership spelling from ADR 0068:

```luau
async function loadAll(urls: Array<Uri>, cancel: CancelToken): Result<List<Page>, Http.Error>
    return await Task.group(cancel, async function(group: Task.Group): Result<List<Page>, Http.Error>
        return await Task.collect(group, urls, Http.get)
    end)
end
```

Bounded channels make backpressure the default:

```luau
local channel: Channel<Job> = Channel.bounded(64)
local sent: Task<Result<(), Channel.Error>> =
    Channel.send(channel, job, cancel)
local next: Task<Result<Job, Channel.Error>> =
    Channel.receive(channel, cancel)
```

Durable scheduling is separate from timers:

```luau
local job = Schedule.every(Time.minutes(15), Jobs.refreshIndex)
local scheduled: Result<Schedule.Id, Schedule.Error> =
    Schedule.add(schedule, job, { overlap = Schedule.Overlap.Skip })
```

## Processes, IPC, terminal, and archives

```luau
local child: Result<Process.Child, Process.Error> = Process.spawn({
    executable = tool,
    arguments = { "check", inputPath },
    input = Process.Input.None,
    output = Process.Output.Capture { maxBytes = 4 * 1024 * 1024 },
}, processCapability)

Terminal.writeLine(terminal, Text.format("checked {0}", { inputPath }))
```

Typed local IPC carries declared messages rather than a property bag:

```luau
local pipe: Process.Ipc<WorkerMessage> =
    Process.openIpc(worker, WorkerMessageSchema, ipcCapability)
local result = Process.send(pipe, WorkerMessage.Build(project), cancel)
```

Archive extraction requires explicit destination authority and limits:

```luau
local result = Archive.extract(input, destination, files, {
    maxEntries = 10_000,
    maxBytes = 2 * 1024 * 1024 * 1024,
    links = Archive.Links.Reject,
})
```

## HTTP client and server

The default request is concise; options expose redirects, pooling, timeout, and
body limits without a builder:

```luau
local request = Http.get(url)
local options = {
    cancel = cancel,
    deadline = Time.after(clock, Time.seconds(10)),
    maxBytes = 4 * 1024 * 1024,
    redirects = Http.Redirect.Safe,
}
local response: Task<Result<Http.Response, Http.Error>> = Http.send(request, options)
```

Routes are typed values and handlers are ordinary functions:

```luau
local routes = Http.routes({
    Http.route(Http.Method.Get, "/users/{id}", showUser),
    Http.route(Http.Method.Post, "/users", createUser),
})
local server = Http.serve(listener, routes, { cancel = cancel })
```

Route-pattern syntax and generated parameter binding require a focused HTTP
metadata proposal; they may not become runtime string dispatch.

## SQL and storage

Parameters and row adapters are generated typed contracts. The checked SQL
literal in this block is proposed syntax and requires a separate language ADR:

```luau
public function findUser(database: Sql.Pool, id: UserId): Result<User?, Sql.Error>
    return Sql.one(database, UserRow,
        sql"SELECT id, name FROM users WHERE id = {id}")
end
```

Without that syntax, generated query values provide the same static contract.
Transactions use a resource scope:

```luau
return Sql.transaction(database, Sql.Isolation.Serializable,
    function(transaction: Sql.Transaction): Result<Order, OrderError>
        match Orders.insert(transaction, draft)
        when Result.Ok(order) then
            match Inventory.reserve(transaction, order.items)
            when Result.Ok(_) then
                return Result.Ok(order)
            when Result.Error(error) then
                return Result.Error(OrderError.Inventory(error))
            end
        when Result.Error(error) then
            return Result.Error(OrderError.Insert(error))
        end
    end)
```

## Messaging and RPC

```luau
local envelope = Message.pack(OrderPlacedSchema, orderPlaced)
local published: Task<Result<Message.Id, Message.Error>> =
    Message.send(broker, Topics.orders, envelope, cancel)
```

Generated RPC bindings retain ordinary typed signatures:

```luau
local call: Task<Result<User, UserRpc.Error>> =
    UserRpc.find(channel, { id = userId }, { cancel = cancel })
```

The broker/channel values identify real protocol resources. Schemas and method
identities are generated at compile time, never looked up from runtime strings.

## Telemetry

Structured data is typed by the event declaration; a sink is explicit:

```luau
@Telemetry.Event(name = "requestComplete")
public record RequestComplete
    method: Http.Method
    status: Http.Status
    elapsed: Time.Duration
end

Telemetry.log(telemetry, RequestComplete {
    method = request.method,
    status = response.status,
    elapsed = elapsed,
})
```

```luau
return Telemetry.span(telemetry, Operations.saveOrder,
    function(span: Telemetry.Span): Result<Order, SaveError>
        Telemetry.field(span, Fields.customerId, customer.id)
        return saveOrder(customer)
    end)
```

## Cryptography and identity

Safe operations select accepted algorithms by typed operation, not arbitrary
strings:

```luau
local digest: Crypto.Hash = Crypto.sha256(payload)
local passwordHash = Crypto.Password.hash(password, Crypto.Password.Argon2id)
local valid = Crypto.Password.verify(password, passwordHash)
local sealed = Crypto.Aead.seal(key, nonce, plaintext, associatedData)
```

Certificate validation is part of connection setup by default. Disabling it is
available only in an explicit unsafe/test transport, never a boolean option on
ordinary `Http.send`.

## Command metadata and settings

```luau
@Command(name = "serve")
@Description(AppMessages.serveDescription)
public record Serve
    @Parameter(short = "p")
    port: UInt16 = 8080
    root: Path
end

public function run(command: Serve, cancel: CancelToken): Result<(), AppError>
    match Settings.load(settingsSources, AppSettingsSchema)
    when Result.Ok(settings) then
        return serve(command, settings, cancel)
    when Result.Error(error) then
        return Result.Error(AppError.Settings(error))
    end
end
```

Attributes are typed compile-time values under the existing UDA contract. The
command/settings processors generate typed adapters and diagnostics; they do not
scan runtime metadata.

A custom metadata type uses existing attribute syntax and a resolved validator:

```luau
@AttributeUsage(targets = { AttributeTarget.Function }, repeatable = false)
@AttributeValidator(validatePermission)
public attribute Permission(name: PermissionName)

@CompileTime
private function validatePermission(name: PermissionName): Boolean
    return PermissionName.isKnown(name)
end
```

## Images and media

```luau
return File.use(inputPath, File.Mode.Read, files,
    function(file: File.Handle): Result<Image.Image, Image.Error>
        match Image.decode(file, { maxPixels = 40_000_000 })
        when Result.Ok(image) then
            return Image.resize(image, { width = 320, height = 180 })
        when Result.Error(error) then
            return Result.Error(error)
        end
    end)
```

Advanced pipelines make frame ownership and backpressure explicit:

```luau
local frames = Video.decode(media, { pool = framePool, cancel = cancel })
local scaled = Video.map(frames, scaler, outputPool)
local result = Video.encode(scaled, output, encoderOptions)
```

## UI

Views are data produced by functions; state transitions are ordinary typed
functions:

```luau
public function view(model: Model): Ui.View<Message>
    return Ui.column({
        Ui.text(model.title),
        Ui.button(Messages.save, Message.Save),
        Ui.list(model.items, itemView),
    })
end

public function update(model: Model, message: Message): Ui.Update<Model, Effect>
    match message
    when Message.Save then
        return Ui.effect(model, Effect.Save(model))
    when Message.Select(id) then
        return Ui.ready(select(model, id))
    end
end
```

Exact UI effect/runtime integration remains an official-package design, but the
example fixes the non-inheritance, typed-message shape.

## Mathematics and science

```luau
local mean: Float64 = Statistics.mean(samples)
local fitted: Result<Statistics.Model, Statistics.Error> =
    Statistics.fit(features, labels, options)

local matrix: Tensor<Float32, Shape<Rows, Columns>> = Tensor.from(values, shape)
local output = Tensor.multiply(matrix, weights, workspace)
```

Streaming statistics avoid materializing the input, while tensor workspaces and
device transfers remain explicit:

```luau
local state: Statistics.Stream.Mean<Float64> = Statistics.Stream.mean()
Sequence.fold(readings, state, Statistics.Stream.add)
local result = Statistics.Stream.finish(state)
```

## AI

```luau
match Ai.load(modelFile, ModelContract, runtime, device)
when Result.Ok(model) then
    local result = Ai.run(model, input, { cancel = cancel })
when Result.Error(error) then
    report(error)
end
```

```luau
local request = Ai.Generate.request(prompt, {
    output = InvoiceSchema,
    maxTokens = 1024,
})
local stream: Task<Result<Ai.Generate.Stream<Invoice>, Ai.Error>> =
    Ai.Generate.start(model, request, credentials, cancel)
```

Model/runtime/credential values are explicit. Structured output and tool calls
use generated schemas and resolved identities, never arbitrary JSON plus string
function names.

## Tooling and native bindings

Tooling consumes stable public values rather than compiler arenas:

```luau
local edit = Source.replace(span, replacement)
local fix = Diagnostic.fix(Diagnostics.invalidName, { edit })
Diagnostic.emit(diagnostics, fix)
```

Generated FFI calls look ordinary but retain explicit native ownership and
error contracts in metadata:

```luau
using Zlib = Native.Zlib

local output: Result<Bytes, Zlib.Error> =
    Zlib.compress(input, buffer, Zlib.Level.Default)
```

The binding generator, not runtime name lookup, resolves `Native.Zlib`. ABI,
target, copy, callback, GC-root, and library-lifetime facts are checked against
the generated binding manifest.

## Tests

```luau
@Test
public function decodingRejectsDeepInput(): Result<(), Test.Error>
    local result = Json.decode(deepInput, UserSchema, { maxDepth = 8 })
    Test.equal(result, Result.Error(Json.Error.DepthLimit))
    return Result.Ok(())
end

@Test.Property(cases = 1000)
public function encodeRoundTrips(user: User): Result<(), Test.Error>
    match Json.encode(user)
    when Result.Ok(encoded) then
        Test.equal(Json.decode(encoded, UserSchema), Result.Ok(user))
        return Result.Ok(())
    when Result.Error(error) then
        return Result.Error(Test.Error.Unexpected(error))
    end
end
```

Test clocks, files, transports, random states, telemetry capture, snapshots, and
fuzz inputs are explicit test values. Production code need not depend on the
test Package.
